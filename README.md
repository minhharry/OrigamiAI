# OrigamiAI
AI model for creating origami crease pattern from 3D model.

## Setup
This project use Python 3.12.11.

Install [uv](https://github.com/astral-sh/uv) package manager.
Install CUDA 12.6 for pytorch.


Then run 
```bash
uv sync
```

## Usage
```bash
.venv/Scripts/activate
python main.py
```
